# Platform Authenticator version support

This guide is an overview of the Beyond Identity Platform Authenticator version support policy. The points below detail the Beyond Identity support plan.

- Certification for the most recent three versions 
- Support for the most recent five versions
- Any older versions than the most recent five are retired

## Retired versions

Any retired Platform Authenticator version cannot authenticate users. When a retired Platform Authenticator launches, users are prompted to update to the most recent version.

:::note
This policy is active on March 1st, 2025.
:::
