Beyond Identity allows you to personalize the look and feel of email communications through a variety of branding options. By customizing these elements, you can maintain consistency with your brand's aesthetic and messaging, reinforcing your identity throughout the user experience.

## Accessing Branding Options

To access the branding options:

1. Log in to the admin console.
2. Navigate to "Deployment".
3. Click on "Branding".

Here, you can view and edit your branding settings.

## General Settings

### Logo

- **URL**: Provide a URL to your custom logo. If no URL is provided, the default Beyond Identity logo will be used.

</p>

### Organization Name

- **Default**: Beyond Identity
- If no name is provided, Beyond Identity will be used as the default name across emails.

### Support Link

- **URL**: [https://www.beyondidentity.com/support](https://www.beyondidentity.com/support)
- If no URL or email is provided, the default Beyond Identity support page will be used.

## Enrollment Email Template

Customize the appearance and content of the enrollment email sent to users.

- **Subject**: Customize the email subject. Example: "Welcome to Beyond Identity".
- **Button Color**: Set the hexadecimal color code for buttons (e.g., #e41150).
- **Button Text Color**: Set the hexadecimal color code for button text (e.g., #FFFFFF).
