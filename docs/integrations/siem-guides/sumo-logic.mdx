## Overview

This guide is for setting up Beyond Identity events data with Sumo Logic.

## Prerequisites

Confirm you have the following prerequisites before continuing with the integration.

- Beyond Identity
    - Tenant configured for organization with ability to enroll users
- Sumo Logic
    - Sumo Logic administrator account

## Stage summary

The integration setup consists of the two stages below. Each stage covers the initial set up for continuing the integration process on each respective platform. Follow the guide in linear order to complete the integration.

| Stage | Summary | Sumo Logic | Beyond Identity |
| :---- | :---- | :---- | :---- |
| [I](#stage-i) | Initial configuration | - [Create hosted collector in Sumo Logic](#-create-hosted-collector-in-sumo-logic)<br></br>- [Add HTTP data source](#-add-http-data-source) | - [Add integration in Beyond Identity](#-add-integration-in-beyond-identity) |

## Stage I

The sections in this stage are for the intial setup of the integration between the two platforms.

### ðŸŸ  Create hosted collector in Sumo Logic

The steps in this section are for creatin ga hosted collector.

1. Navigate to your Sumo Logic tenant account login.
2. On the Sumo Logic dashboard, under **Manage Data**, click **Collections**.
3. On the right side top menu, click **Add Collector**.
4. On the **Select Collector Type** screen, click **Hosted Collector**.
5. Input `Tenant events from Beyond Identity` in the **Name** field and add a description on the **Add Hosted Collector** screen.
6. Click **Save**.

### ðŸŸ  Add HTTP data source

This section has the instructions for adding an HTTP data source in Sumo Logic.

1. After creating a hosted collector, the pop up for adding a data source appears.  Click **OK**.
2. Input `HTTP` in the search box and select **HTTP Logs & Metrics** in the search results.
3. Input `HTTP Log source for Beyond Identity Tenant` in the **Name** field and add a description.
4. Maintain the default values for the other fields. Click **Save**.
5. On the **HTTP Source Address** pop up, click **Copy** to copy the value and save it to a secure location. This value is required for SECTION.
6. Click **OK**.

### ðŸ”µ Add integration in Beyond Identity

The section here includes the steps for configuring the SIEM integration in Beyond Identity. The instructions below are for the Beyond Identity Console.

1. On the Beyond Identity Console lefthand navigation, under Platform, select **Integrations**.
2. Select the **Browse integrations** tab.
3. Select **Sumo Logic**.
4. On the Sumo Logic overview page, click **Add instance**.

## Stage II

The sections in this stage are for setting up the Sumo Logic configuration in Beyond Identity and viewing the logs. Follow the sections below to complete the integration.

### ðŸ”µ Configure Sumo Logic in Beyond Identity
