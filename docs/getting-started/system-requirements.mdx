#System requirements
The latest release supports the following operating systems. This list will be updated as additional platforms are added. 
##Supported operating systems
| **Operating System** | **Versions**               | 
|------------------|--------------------------------|
| macOS            | 10.15, 11, 12, 13, 14 (Sonoma) |
| Windows          | 10 (build 18363 and later), 11 |
