The Beyond Identity Authenticator is an app designed to allow an end user to quickly and easily add and manage passkeys, and authenticate to devices and apps configured to authenticate with Beyond Identity.

Install the base authenticator from:

https://app.byndid.com/downloads 

Install the Authenticator for Windows Desktop Login from:

https://app.byndid.com/desktop-login/downloads
